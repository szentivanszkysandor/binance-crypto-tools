Privacy Policy

Effective Date: September 24, 2025

1. Overview

The “Binance – Crypto Tools” Chrome extension provides a unified interface for monitoring live Binance crypto prices and using built-in HTML5 trading tools. It displays real-time Price, RSI(14), and 24h% updates (plus EMA Trend), and includes six optional tools like Order Book, Market Trades, and Macro/Crypto Risk dashboards.

The extension itself does not trade, manage wallets, or access private Binance accounts. It is an independent, unofficial project and is not affiliated with Binance. All data is purely informational and intended for educational and analytical purposes.

2. Permissions Used

  • storage: Used to save your configuration locally, including:
    • Your selected crypto symbols and their timeframes.
    • Interface preferences such as hiding tool buttons, hiding the Live Prices section, enabling Compact View, and column visibility choices.
    • Minor UI state like onboarding tip dismissal and Compact View toggle state.
  • Host permission: Required to connect securely to the official Binance REST API:
    • https://api.binance.com/* — strictly used to fetch public, read-only market data (prices, klines for RSI/EMA/MACD/ATR, 24h stats).
    • No private account data, authentication, or trading features are accessed.

The extension does not request access to browser tabs, browsing history, or other sensitive permissions.

2.1 Clipboard Use

When you click “Copy address” in the donation modal, the extension writes the displayed BNB (BEP20) address to your clipboard. It does not read clipboard contents and does not access the clipboard without your direct click.

3. Data Storage and Use

  • User settings: Stored locally in chrome.storage.sync. If Chrome Sync is enabled, these settings may sync with your Google account under Google’s safeguards.
  • No personal data: The extension does not collect email addresses, passwords, wallet keys, balances, or browsing history.
  • No external transmission: Aside from anonymous requests to the Binance API over HTTPS, no data is sent to any external server by the extension.

4. Binance API Data

  • The extension fetches public, anonymous market data directly from Binance endpoints, including:
    • Live symbol prices
    • Candlestick data (for local RSI/EMA/MACD/ATR calculations)
    • 24-hour change & volume statistics
  • No Binance account login, API key, or authentication is required or used.
  • All requests are read-only and contain no personal identifiers.

5. Third-Party Services

  • Binance APIs: Market data comes directly from Binance. Please refer to Binance’s privacy policy for details on their services.
  • Developer-hosted tool pages: The “Tools” buttons open standalone, static HTML pages hosted by the developer (e.g., on GitHub Pages). These pages are separate from the extension’s UI and do not receive any personal data from the extension. They may be subject to the hosting provider’s own policies.
  • Donation links: Optional donation buttons (Binance QR and PayPal) open external payment pages. No payment data is collected or stored by this extension.

6. No Tracking or Analytics

  • No telemetry or behavior tracking is implemented.
  • No cookies or fingerprinting are used.
  • The extension never injects scripts into third-party web pages or read their content.

7. Incognito Mode

The extension may run in Incognito mode if manually enabled by the user in Chrome. Its privacy behavior is identical to normal mode.

8. Security

  • All data remains in Chrome’s secure local storage.
  • All external requests use HTTPS.
  • The extension has no backend servers and does not handle sensitive data such as login credentials or payment information.
  • No remote code execution; the extension does not fetch or execute code from external sources.

9. Data Retention & Deletion

  • Retention: Your symbol list and preferences remain stored locally until you edit or reset them.
  • Manual management: Use the Options page to modify or clear your configuration at any time.
  • Uninstall: Removing the extension automatically deletes all its stored data from Chrome.

10. Changes

If this privacy policy changes due to feature updates or permission changes, the latest version will always be available through the Chrome Web Store listing.

11. Contact

For support or privacy-related concerns, please use the contact information provided in the Chrome Web Store listing.

12. Children’s Privacy

This extension is not directed to children under 13. It does not knowingly collect personal information from children.

13. No Financial Advice

The extension provides market data and tools for informational and educational purposes only. It does not constitute financial, investment, or trading advice. You are solely responsible for your trading decisions.

This extension complies with the Chrome Web Store Developer Program Policies and Google’s User Data policies. It collects no personal information and shares nothing with third parties.
