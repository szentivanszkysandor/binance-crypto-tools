Privacy Policy

Effective Date: September 23, 2025

1. Overview

The “Binance – Crypto Tools” Chrome extension provides a unified interface for monitoring live Binance crypto prices and using built-in HTML5 trading tools. It displays real-time Price, RSI(14), and 24h% updates, and includes six optional tools like Order Book, Market Trades, and Macro/Crypto Risk dashboards.

The extension itself does not trade, manage wallets, or access private Binance accounts. It is an independent, unofficial project and is not affiliated with Binance. All data is purely informational and intended for educational and analytical purposes.

2. Permissions Used

  • storage: Used to save your configuration locally, including:
    • Your selected crypto symbols and their timeframes.
    • Interface preferences such as hiding tool buttons or enabling compact view.
  • tabs: Allows opening Binance.com or selected tools in new browser tabs.

No content scripts or analytics permissions are included. The extension requests https://api.binance.com/* read-only access solely for live market data.

3. Data Storage and Use

  • User settings: Stored locally in chrome.storage.sync for convenience. These never leave your browser unless you enable Chrome Sync, in which case they sync securely with your Google account.
  • No personal data: The extension does not collect email addresses, passwords, wallet addresses, balances, or browsing history.
  • No transmission: The extension itself does not send any information to external servers other than the public Binance API for market data.

4. Binance API Data

  • The extension fetches public, anonymous market data directly from Binance endpoints, including:
    • Live symbol prices
    • RSI calculations based on recent candlestick data
    • 24-hour percentage change
  • No Binance account login, API key, or authentication is required or used.
  • All requests are read-only and contain no personal identifiers.

5. Third-Party Services

  • GitHub Pages: Optional tools opened via the extension may be hosted on GitHub or other sites. Their terms and privacy policies apply.
  • Binance APIs: Market data comes directly from Binance. Please refer to Binance’s privacy policy for details on their services.
  • Donation links: Static links (Binance QR or PayPal) may be provided for optional support. These open external payment services; no payment data is collected or stored by the extension.

6. No Tracking or Analytics

  • No telemetry or behavior tracking.
  • No cookies or fingerprinting.
  • No access to page content, clipboard, or other browser data beyond what is required for the extension’s core functions.

7. Incognito Mode

The extension may run in Incognito if manually enabled by the user in Chrome. Functionality and privacy behavior are identical to normal mode.

8. Security

All data remains in Chrome’s secure storage. The extension has no backend servers and does not handle sensitive data such as login credentials or payment details.

9. Data Retention & Deletion

  • Retention: Your symbol list and settings remain until you edit or reset them.
  • Manual management: Use the Options page to modify or clear your configuration at any time.
  • Uninstall: Removing the extension automatically deletes all its stored data from Chrome.

10. Changes

Updates to features or permissions may require changes to this policy. The latest version will always be available through the Chrome Web Store listing.

11. Contact

For support or privacy concerns, please use the contact form provided in the Chrome Web Store listing.

This extension complies with the Chrome Web Store Developer Program Policies and Google’s user data policies.
